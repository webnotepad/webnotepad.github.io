# WebNotepad - Online Notepad & Productivity Tools

**Live Demo**: [https://webnotepad.github.io](https://webnotepad.github.io)

A clean, fast, and fully browser-based notepad and productivity suite. Write notes, journal, brainstorm with mind maps, track habits, and more — all without signup, downloads, or server dependency. Your data stays private in your browser's local storage.

## ✨ Features

### Core Notepad
- **Instant Auto-Save**: Every keystroke is automatically saved to local storage.
- **Rich Text Editing**: Bold, italic, underline, strikethrough, lists, text alignment, font sizes, and color picker.
- **Multiple Notes**: Create, organize, search, and switch between unlimited notes with custom titles.
- **Find & Replace**: Powerful inline search and replace functionality.
- **Live Statistics**: Real-time word count, character count, line count, and estimated reading time.
- **Dark Mode**: Eye-friendly theme toggle.
- **Fullscreen Mode**: Distraction-free writing environment.
- **Export Options**: Download notes as clean `.txt` or formatted `.html` files.
- **Offline Support**: Works completely offline once loaded.
- **Responsive Design**: Seamless experience on desktop and mobile.

### Additional Productivity Tools
- **Diary**: Private dated journal entries with rich text support.
- **MindMap**: Interactive mind mapping for brainstorming and idea visualization.
- **List Maker**: Drag-and-drop checklists and to-do lists.
- **Case Converter**: Quickly transform text between uppercase, lowercase, title case, etc.
- **Random Text Generator**: Generate placeholder content for testing or drafting.
- **Word Shuffler**: Randomize word order for creative writing prompts.
- **Pomodoro Timer**: Focus timer with customizable work/break intervals.
- **Habit Tracker**: Visual calendar to build and track daily habits.
- **Decision Maker / Choice Maker**: Tools like coin flips or wheels for quick decisions.

### Other Highlights
- Zero data collection or tracking of your notes.
- Blazing fast performance with a minimalist, distraction-free UI.
- Built for students, writers, professionals, developers, and everyday users.

## 🚀 Quick Start

1. Visit [https://webnotepad.github.io](https://webnotepad.github.io)
2. Start typing — your notes are saved automatically.
3. Use the sidebar to manage multiple notes.
4. Explore other tools via the navigation.

No installation or account required.
